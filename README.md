# How to install
```bash
  curl -Lks https://raw.githubusercontent.com/weslley39/dotFiles/master/install.sh | /bin/bash
```
# How to push changes
```bash
config status
config add .vimrc
config commit -m "Add vimrc"
config add .bashrc
config commit -m "Add bashrc"
config push
```
